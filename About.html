About jalanMY

Learn more about the purpose, inspiration, and audience behind this Kuala Lumpur travel guide.

Website Background

jalanMY is a simple travel guide website created to showcase popular attractions, cultural hotspots, and experiences in Kuala Lumpur. The website was developed as part of an educational project to demonstrate the use of HTML and CSS in building a structured and visually appealing multi-page website.

Kuala Lumpur is a diverse city filled with modern landmarks, traditional culture, and vibrant street life. This website aims to highlight those elements in a clear and easy-to-understand format.

Purpose of the Website

The main purpose of jalanMY is to provide users with a quick and simple overview of key attractions and experiences in Kuala Lumpur. Rather than overwhelming users with too much information, the website focuses on clarity, ease of navigation, and visual presentation.

This project also serves as a learning platform to apply web design principles such as consistent layout, responsive design, and user-friendly navigation.

Target Users

  • Tourists visiting Kuala Lumpur for the first time
  • Local visitors looking for weekend activities
  • Anyone interested in exploring KL’s culture and attractions
